# Simple_shop

## 簡介
* 我們製作了會員管理、產品管理、購買記錄管理、供應商管理以及查詢。
* 查詢功能並沒有修改和刪除的選項，而是純粹的查詢。
* 另外前四個管理功能上，我們假設只需要輸入Primary Key即可新增，其他屬性是否應該輸入的情況我們一律不考慮。
